# 🌍 Travel Bucket List Website - Wander Goals

> Final Assignment for the Web Design for Beginners Course conducted by the University of Moratuwa Open Learning Platform.

---

## 📖 About the Project

This project was developed as the **final assignment** for the **Web Design for Beginners** course offered through the Open Learning Platform of the University of Moratuwa.

The objective of the assignment was to design and develop a simple, interactive website using **HTML, CSS, and JavaScript** while applying fundamental web development concepts learned throughout the course.

For this assignment, I created a **Travel Bucket List Website** that showcases dream destinations I hope to visit in the future. The website provides an engaging user experience through responsive design, attractive styling, and interactive elements.

---

## ✨ Features

* 🗺️ Display of travel destinations on my bucket list
* 📸 Destination images and descriptions
* 🎨 Modern and responsive user interface
* 📱 Mobile-friendly design
* ⚡ Interactive elements using JavaScript
* 🧭 Easy navigation and clean layout
* 🌍 User-friendly travel-themed design
